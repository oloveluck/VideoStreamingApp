# VideoStreamingApp
TV App for CS3200: Database Design at Northeastern University

# Backend API
To setup the Video Streaming App API:
  1. Install the latest version of Node
  2. pull this repo
  3. navigate to /backend
  4. run npm install
  5. start the api on localhost:8080 with npm start or node app.js
