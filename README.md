# VideoStreamingApp
TV App for CS3200: Database Design at Northeastern University

# Backend API
To setup the Video Streaming App API:
  1. Install the latest version of Node
  2. pull this repo
  3. navigate to /backend
  4. run npm install
  5. start the api on localhost:8080 with npm start or node app.js
  
# Database 
To setup the MariaDB Database management system :
  1. Using XAMPP run MariaDB and PHPMyAdmin with Apache 
  2. Create a database called "yatva"
  3. Import the DDL.ql then the DML.sql files.
  
# Frontend
  1. Host yatva.html on localhost:8000 (I use the command python -m http.server)
